This repository contains the datasets used in the submitted manuscript to Expert Systems with Applications titled 
"Enhancing Multilingual Retrieval-Augmented-Generation Systems: Bridging the Gap for Turkish Language and Medical Domain Applications"

* The file "pubmed-rag-tr.csv" contains the synthetic dataset generated for the work.
* Files "llm-as-judge-eval-data-part1.csv" and "llm-as-judge-eval-data-part2.csv" contain the LLM-as-a-judge dataset evalaution results splitted into 2 halfs due to upload limit.
